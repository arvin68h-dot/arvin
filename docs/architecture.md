## 模块说明

### CLI
用户交互入口。支持命令行模式和 ACP 协议模式。
- `main.ts` — 命令路由
- `commands/` — 各子命令实现
- `tui/` — 终端交互界面
- `acp-host.ts` — CodeWorker 集成网关

### Core
核心类型定义和基础服务。
- `types/` — 所有类型定义 (AgentMessage, Provider, Tool, etc.)
- `config/` — 配置系统
- `provider/` — LLM Provider 实现
- `util/` — 工具函数 (日志等)
- `protocol/` — ACP 协议定义

### Tool
工具注册和执行系统。
- `file/` — 文件操作工具
- `shell/` — Shell 命令执行
- `search/` — 代码搜索
- `tool_registry.ts` — 工具注册表

### Engine
各语言特定的构建和编译引擎。
- 每种语言一个模块
- 继承基类，实现 build/format/lint 接口

### Storage
SQLite 本地存储。
- 会话数据
- 消息记录
- 快照管理

### Checkpoint
快照系统。
- 创建/列出/恢复/比较快照

### Task
任务拆解与调度。
- 任务规划
- 任务调度
- 任务追踪

### MCP
MCP 通信支持。
- JSON-RPC 服务器/客户端

## 关键技术决策

1. **pnpm Monorepo** — 13 个核心包 + CLI，TypeScript composite build
2. **tsx 运行时** — 直连源码运行，规避 Node.js ESM Loader 限制
3. **bundler 模块解析** — 使用 paths 映射实现 Monorepo 跨包引用
4. **SQLite** — 轻量级本地存储，无需外部数据库服务
5. **string enum** — 避免 TypeScript enum 索引访问限制
6. **异步 IIFE** — 规避 Node.js ESM 顶层 await 限制
